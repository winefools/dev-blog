ChatGPT의 말:

Stack 테마에서 글 포스트를 만들려면 Hugo의 new 명령을 쓰면 됩니다.

  1. 새 글 생성

cd ~/dev-blog hugo new posts/first-post.md

이렇게 하면 content/posts/first-post.md라는 Markdown 파일이 생깁니다. 2. 글 작성

생성된 파일을 열어 내용 작성:

title: “First Post” date: 2025-08-13T18:00:00+09:00 draft: true

여기에 본문을 작성하세요.
코드 블록 예시:

echo "Hello Hugo"


> `draft: true` 상태면 기본 Hugo 서버에서만 보이고, 배포 시에는 안 나옵니다.  
> 배포하려면 `draft: false`로 바꾸거나 해당 줄을 삭제하세요.

---

## 3. 로컬 확인
```bash
hugo server -t stack

→ 브라우저에서 http://localhost:1313/posts/first-post/ 접속
4. 목록 페이지에서 보이게 하기

Stack 테마는 보통 content/posts 안의 Markdown들을 자동으로 리스트 페이지(/posts/)에 모아줍니다.

원하면 제가 Stack 테마 스타일에 맞춰서 개발일지 전용 포스트 템플릿을 만들어 드릴 수 있어요.
그럼 hugo new로 만들면 자동으로 제목, 날짜, 태그, 코드블록 스타일까지 들어가게 됩니다.
